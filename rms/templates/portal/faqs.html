Frequently Asked Questions

1. Can we book a resource on today's date

Resources management system does not allow booking for the current date to be done on current date. You can only book resources from the day next to the current date.

2. Can anyone register for Resources management system

Since Resources management system is built with the sole purpose to manage various resources of Computer Science and Engineering department of National Institute of Technology, only Faculty, staff and students are allowed to do registration. However students need to get their registration approved from the admin so that they can start using the system.
